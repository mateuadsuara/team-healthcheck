# team-healthcheck

data visualisation tool for assessing team health

# get started

* install elm
* install ruby

### install elm

`npm -g install elm`
`npm -g install elm-test`

more info: https://guide.elm-lang.org/install.html

### install ruby

more info: https://www.ruby-lang.org/en/documentation/installation/

### run frontend tests

`elm-test tests/`

### run backend tests

`rspec spec/`

### compile frontend

`elm-make src`

### start backend

`rackup`
